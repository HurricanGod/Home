**数组**：

定义数组的方法：

+ `var array = [1,2,3]`
+ `var array = []`
+ `var array = new Array()`

**注意**：

+ **Javascript**的数组长度是可变的
+ 同一个数组里的元素类型可以不一致
+ 访问数组不会出现数组越界，访问未赋值的数组元素时，元素的值为 `undefined`
